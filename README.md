# CSV Data Analyser

A simple Python project that reads a CSV file, analyses its structure and quality, removes duplicate rows, and exports cleaned data and a text report.

## Features

- Reads CSV data using pandas
- Displays column names
- Shows row and column counts
- Counts missing values
- Detects duplicate rows
- Removes duplicates
- Exports cleaned CSV
- Generates a text-based analysis report

## How to Run

```bash
py analyser.py
